Which of the following are the features of JSON? 1 Interoperability 2 Simplicity 3 Extensibility 4 All of the above
1 / 1

All of the following are features of JSON:

  1. All of the above

  2. Interoperability: JSON is a widely adopted data interchange format that is supported by many programming languages and platforms. This makes it easy to share data between different systems regardless of their underlying technologies.

  3. Simplicity: JSON has a simple and easy-to-understand syntax. It uses a combination of key-value pairs, objects, and arrays to represent structured data in a straightforward manner.

  4. Extensibility: JSON supports nested structures, allowing you to represent complex data hierarchies. You can create your own custom data structures by combining objects and arrays, making it an extensible format for various data representation needs.

JSON's popularity has largely grown due to these features, making it a convenient choice for data interchange between different applications and systems.
